## List of Subroutines 
The PS4 folder contains the following files/subroutines
More info on how the subroutine works is commented in the respective files

- aiyagari.jl     : calculates the policy function for aiyagari model 
- exp_grid.jl     : gives exponential increasing grid points b/w interval
- FE_test.jl      : simple test case for finite element methods
-linear_basis.jl  : calculates linear basis fns given grid points
- multivar_newt_root.jl :subroutine to calculte zero using newton method
- num_quadrature.jl    : subroutine to calculate integrals using numerical quadrature
- plots.jl          : script to get the plots
        

## How to get the results

- Run FE_test.jl to get results for the basic case
- Run plots.jl to get plots
- Run aiyagari.jl to get results for aiyagari model
- Run plots.jl to get plots
